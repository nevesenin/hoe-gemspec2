= hoe-gemspec

{Project}[http://rubygems.org/gems/hoe-gemspec]
{Documentation}[http://libraggi.rubyforge.org/hoe-gemspec/README_rdoc.html]
{Wiki}[http://wiki.github.com/raggi/hoe-gemspec/]
{Source Code}[http://github.com/raggi/hoe-gemspec/]
{Issues}[http://github.com/raggi/hoe-gemspec/issues]
{Rubyforge}[http://rubyforge.org/projects/libraggi]

== DESCRIPTION:

Adds support for generation of gemspec files to Hoe.

== FEATURES/PROBLEMS:

* task: gemspec
* task: default enhanced with gemspec

== SYNOPSIS:

  Hoe.plugin :gemspec

== REQUIREMENTS:

* Hoe

== INSTALL:

* gem install hoe-gemspec
